# Meetings

This directory stores meeting notes and discussion summaries that should remain tied to the repository.

## Suggested Naming

- `YYYY-MM-DD.md` for a single meeting or discussion summary
- `YYYY-MM-DD-topic.md` when multiple notes exist on the same day

## Suggested Structure

- Context
- Participants
- Key decisions
- Open questions
- Action items
- Follow-up date
